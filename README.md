# food-ordering

Monorepo for the food-ordering platform.

## Workspace

- Package manager: `pnpm`
- Apps: `auth`, `catalog`, `order`, `product`, `shell`
- Shared packages live under `packages/`

## Quick Start

```bash
pnpm install
```

## Notes

- This repository is configured as a pnpm workspace via `pnpm-workspace.yaml`.
- Add per-app scripts in each app package when needed.
